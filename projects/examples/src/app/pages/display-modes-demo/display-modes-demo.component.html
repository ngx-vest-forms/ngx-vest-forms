Display Modes Demo

This demo showcases the new 'on-dirty' and 'always' display modes for errors and warnings.

Error Display Mode: 'always'

Errors appear immediately, even on pristine fields. Notice the error shows up without any interaction.

Error Display Mode: 'on-dirty'

Errors appear as soon as you type (field becomes dirty). No need to blur the field first.

Warning Display Mode: 'always'

Warnings appear immediately, even on pristine fields. Type less than 5 characters to see the warning.

Warning Display Mode: 'on-dirty'

Warnings appear as soon as you type (field becomes dirty). No need to blur first.

Warning Display Mode: 'on-touch' (for comparison)

Warnings appear only after you blur the field (traditional behavior).
