Cyclic Dependencies Form Example

This form demonstrates how ngx-vest-forms handles cyclic dependencies between fields using the validationConfig input.

  • The 'Amount' field is required if 'Description' is not empty.
  • The 'Description' field is required if 'Amount' is not empty.

The library ensures that such configurations do not lead to infinite validation loops.
